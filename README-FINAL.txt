🚀 PRÓXIMOS PASOS

PASO 1: Descargar Android Studio
  → https://developer.android.com/studio

PASO 2: Generar APK
  → npm run apk:debug

PASO 3: Instalar en tu teléfono
  → adb install -r android/app/build/outputs/apk/debug/app-debug.apk

PASO 4: ¡Probar la app en tu dispositivo!
