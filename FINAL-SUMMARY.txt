   npx cap sync android
   cd android
   ./gradlew assembleDebug

INSTALAR EN TELÉFONO:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Conecta tu teléfono Android por USB
2. Activa "Depuración USB" en Configuración > Opciones de desarrollador
3. Opción A - Script:   .\build-apk.ps1 → Opción 4
   Opción B - Manual:   adb install -r android/app/build/outputs/apk/debug/app-debug.apk
   Opción C - Studio:   npm run mobile:open → Click Run
